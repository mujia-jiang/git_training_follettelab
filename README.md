# git_training_follettelab
Repo used for training students to use Git Version Control and GitHub.

Welcome! We're going to go through a few training tasks to get the hang of Git. We'll also practice our **markdown** `skills` _while we're at it_!

1. If you don't already have one, go ahead and make a GitHub account.
2. Start out by `forking` this repo to your own GitHub account. This makes your own copy which you can control, make edits to, and then connect your changes back to the original repository or "repo".
3. `Clone` the forked repo to your computer (_Advanced version: clone this repo, and **then** add your fork of it as a different remote._) Your `upstream` branch should be your own fork.
4. Explore the repo locally, see what's in it! Use the CLI and the GUI/"finder" app.
5. Find and run the python file in the repo. Does it make any output files? Does git "see" those files? Figure out why/not!
